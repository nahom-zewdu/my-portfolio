# Portfolio Vision

## Purpose

The portfolio should communicate:

1. I build difficult things.
2. I know what I'm doing.
3. I think deeply about systems.

## What it is NOT

- Startup landing page
- Personal blog
- Designer portfolio
- AI-themed website
- Terminal-themed website

## Design Language

Engineering Notebook

The site should feel like reading the notes of a systems engineer.

Keywords:

- Editorial
- Structured
- Intentional
- Calm
- Technical
- Confident

Avoid:

- Flashy animations
- Glassmorphism
- Gradients
- Neon colors
- Excessive motion
- Bento grids

## Layout

Persistent left navigation.
Scrollable content area.

## Typography

Typography is the primary visual identity.

Large headings.
Generous whitespace.

## Accent Color

Muted amber / brass / aged yellow.

Used sparingly.
